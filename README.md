#  Caixa de Recados

### Comandos:
- git clone https://github.com/HickSouldrow/LARAVEL-CRUD.git
- composer i
- npm i
- php artisan key:generate
- copy .env.example .env
- php artisan migrate
- npm run build
- (em outro terminal) php artisan serve
